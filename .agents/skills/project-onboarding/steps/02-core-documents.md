# 核心文档专门了解

## Objective

在已有项目整体认识的基础上，专门阅读指定核心文档，形成项目核心架构理解。

## Resources

None

## Instructions

1. 使用上一步 handoff 作为背景。
2. 查找并阅读“对话持久化与注意力窗口架构 (Transcript Persistence & Attention Window Architecture)”文档。
3. 查找并阅读“统一核心与多接入面 (Unified Core & Access Surfaces)”文档。
4. 查找并阅读“生命周期概念定义与规范 (Lifecycle Concepts)”文档。
5. 阅读 `research/internals/screen-rendering/overview.md`。
6. 如果标题文档不在明显路径下，使用中文标题、英文标题或关键词在仓库中搜索。找不到的文档要明确报告，不要臆造内容。
7. 输出一份核心架构摘要，把这些文档与上一步的项目地图联系起来。

## Output

一份项目核心架构理解摘要。

## Completion Criteria

- 每个指定文档都已阅读，或已明确说明未找到。
- 摘要说明了对话持久化与注意力窗口、统一核心与多接入面、生命周期概念、屏幕渲染之间的关系。
- 最终回答足以帮助用户继续进行开发、审查或调试工作。
- 不需要继续读取下一步。

## Handoff

无需下游 handoff。这是终止 step。
