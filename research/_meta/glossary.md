# 术语表 (Glossary)

> 统一术语定义，消除认知歧义。按字母排序，中英对照。

## A

### Agent Loop（智能体循环）

LLM 与外部工具之间的迭代交互循环。LLM 生成响应（可能包含工具调用），系统执行工具并将结果返回 LLM，LLM 基于结果继续推理，直到任务完成或需要用户输入。

### ADR (Architecture Decision Record, 架构决策记录)

记录重要架构决策的轻量级文档格式，包含背景、决策、依据和影响。

## C

### Context Window（上下文窗口）

LLM 单次推理能处理的最大 token 数量。所有的系统提示、对话历史、工具描述、工具结果都需要在这个窗口内。

## F

### Function Calling（函数调用）

LLM 生成结构化的函数调用指令（而非自然语言）的能力。是 Tool Use 的底层机制。

## M

### MCP (Model Context Protocol)

Anthropic 提出的标准化协议，用于 LLM 应用与外部工具/数据源之间的通信。

## S

### Sandbox（沙箱）

隔离的执行环境，用于安全地运行不受信任的代码或命令，限制其对系统资源的访问。

### System Prompt（系统提示词）

在对话开始时提供给 LLM 的指令，定义其角色、能力、约束和行为规范。

## T

### Tool Use（工具使用）

智能体通过调用预定义的工具（如文件读写、命令执行、网络搜索等）来扩展自身能力的机制。

---

> 本术语表随研究推进持续更新。当在认知研究中遇到新术语时，先在此处建立统一定义。
