# 竞品图谱 (Competitive Landscape)

> 同类产品的系统化对比分析，识别差异化机会

## 分析框架

我们关注的不是功能列表的简单对比，而是：

- **架构选择** — 各家做了怎样的架构取舍？
- **用户体验** — 各家的 UX 设计哲学是什么？
- **差异化点** — 各家独特的竞争优势在哪里？
- **市场空白** — 哪些需求尚未被很好地满足？
