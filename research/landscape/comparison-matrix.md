## 差异化机会

<!-- 从对比中识别的市场空白和差异化方向 -->

1. **自主学习 × 编码深度的交叉地带**：Hermes Agent 有闭环学习但非编码专精，Claude Code 编码强但无自主进化 → 知行可以在编码场景中实现自主学习
2. **TypeScript 全栈优势**：Hermes 选择 Python，在前端渲染、类型安全、打包分发上受限 → 知行用 TS 可同时覆盖 CLI/Web/IDE 场景
3. **架构内聚性**：Hermes 核心文件行数膨胀（单文件 9K+ 行），OpenClaw 模块化程度更高 → 知行追求模块化 + 可维护性的最佳平衡
4. **渐进式复杂度**：现有产品要么功能简单，要么一开始就暴露全部复杂度 → 知行可以做到从简单 CLI 到完整平台的渐进式解锁
5. **透明的技能进化**：Hermes 后台静默创建/更新技能（用户不知情），知行采用"反思提议 + 用户确认"模式，同时具备使用追踪和生命周期治理——在"越用越聪明"方向上超越 Hermes
