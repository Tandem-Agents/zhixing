## 模块说明

设计中心是认知研究的产出物，也是未来代码实现的输入物。它建立在 `insights/` 和 `source-analysis/` 的研究基础之上，每个设计决策都应当可以追溯到具体的认知研究。

## 设计演进原则

1. **研究先行** — 不做没有认知支撑的设计决策
2. **渐进明细** — 先粗后细，允许设计随认知深化而演进
3. **决策可逆** — 标记每个决策的可逆程度，对不可逆决策格外慎重
4. **对标超越** — 每个设计点都明确：我们对标谁？在哪里超越？
