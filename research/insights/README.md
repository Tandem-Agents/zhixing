## 研究规范

- 每个问题文件使用 [`_templates/insight-question.md`](../_templates/insight-question.md) 模板
- 单个文件包含 1-3 个紧密相关的问题，避免过大或过碎
- 文件名使用英文短横线命名，如 `what-is-agent-loop.md`
- 研究发现必须标注信息来源，遵循 [`_meta/sources.md`](../_meta/sources.md) 的可信度标准
- 完成一个问题后及时更新 [`_meta/progress.md`](../_meta/progress.md)
