# 认知域 05：安全模型 (Security Model)

> 理解智能体如何在强大能力与安全约束之间取得平衡

## 领域概述

智能体拥有读写文件、执行命令、访问网络等强大能力，安全模型决定了这些能力何时可用、需要什么级别的授权、以及如何防止误操作和恶意利用。OpenClaw 的 VISION.md 明确表态："strong defaults without killing capability"，这是一个关键的设计哲学。
