# 知行 (Zhixing)

> 知而后行，行而后知 —— 一个为真实工作场景构建的个人智能体

## 项目简介

知行是一个独立部署的个人 AI 智能体，运行在你自己的设备上，遵循你自己的规则。

它不是一个聊天机器人，而是一个能真正执行任务的助手——读写文件、运行命令、搜索信息、管理工作流——在你的完全掌控之下。


## 设计理念

- **本地优先** — 数据和执行都在你的设备上，隐私由你掌控
- **能力与安全的平衡** — 强大的默认能力，明确的安全边界
- **渐进式实现** — 每一步都是独立且可验证的
- **开放扩展** — 核心精简，能力通过插件无限扩展
